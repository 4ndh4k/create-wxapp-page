# create-wxapp-page

## 功能

- 自动化生成微信小程序页面，支持（less, scss, wxss）
- 自动更新app.json,添加生成页面

## 安装

```bash
$ yarn creat-wxapp-page
```


## 用法

```bash
$ cd ./my-wxapp-project
$ create-wxapp-page -d ./src/
#回答问题
#自动生成...
```
